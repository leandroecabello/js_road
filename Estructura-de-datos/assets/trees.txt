Los arboles son una estructura de datos que puede ser lineal o no ( ahora lo vemos), los arboles tienen una estructura 
con nodos que estos representan ser los hijos podemos ver en esta imagen todos los componentes que forman un arbol

Root o Raiz: Es el nodo superior de todo el arbol, si sabes HTML(lo mas probable es que si) seria como en el DOM el HTML 
es el elemento root o más bien el padre
Parent / Padre: Contienen ramas que apuntan a otros nodos
Child / Hijos: Es un nodo conectado directamente con otro cuando se aleja de la raiz
Lead / Hojas: Un nodo sin hijos
Sibling / Hermanos: Son un conjunto de nodos que vienen del mismo padre
Sub Tree / Sub Arbol: Son arboles que estan dentro de nodos
Arm / Baso: Es la conexión entre un nodo y otro
